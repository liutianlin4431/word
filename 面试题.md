## 1、自增变量

```java
public static void main(String[] args) {
		int i = 1;
		// 先执行=右侧运算
		// 将i压入操作数栈
		// i进行加1操作；i从1变成2
		// 执行=赋值操作，将操作数栈中的赋值给i
		// 最终i=1
		i = i++;
		int j = i++;
		int k = i + ++i * i++;
		System.out.println(i);
		System.out.println(j);
		System.out.println(k);
	}
```

```java
结果：
	4
	1
	11
```

```tex
小结：
*赋值“=”，最后算
*“=”右边的从左到右加载值依次压入操作数栈
*实际先算哪个，看运算符优先级
*自增、自减（不使用“=”进行赋值时）操作都是直接修改变量的值，不经过操作站
```
